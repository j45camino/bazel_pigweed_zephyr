# Bazel Pigweed Project

This project demonstrates how to set up a Bazel project with Pigweed as a Git submodule.

## Building

To build the `hello_world` application, run the following command:

```bash
bazel build //apps/hello_world
```

## Running

To run the `hello_world` application, run the following command:

```bash
./bazel-bin/apps/hello_world/hello_world
```
